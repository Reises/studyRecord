# 学習記録用アプリ
学習内容および学習時間を表示できるウェブアプリケーションです。
フロントエンドはReact+JavaScript

# 機能概要
* 学習内容、学習時間を入力と表示
* 学習内容、学習時間の削除
* ローカルストレージに学習内容、学習時間を保存（別のものに変更予定）
